## QA Manual Pro - Advanced Practical Course

Intensive hands-on program focused on transforming theoretical QA knowledge into real project experience.  
Course covered industry-standard documentation practices, Frontend/UI/UX testing, and introduction to Backend & API testing.

### Key Modules
- **Professional QA Documentation:**  
  Bug reports, test cases, checklists, scenarios (ISTQB & IEEE standards)
- **Frontend Testing:**  
  DevTools, responsiveness, browser compatibility, accessibility (WCAG)
- **UI/UX Testing:**  
  Usability principles, user flows, visual testing concepts
- **Backend & API Testing:**  
  API structure, HTTP methods (GET/POST/PUT/DELETE), Postman practice, JSON validation
- **SQL Basics:**  
  SELECT, INSERT, UPDATE, DELETE for test data validation

### Practical Outcomes
- Built real-world QA documentation for portfolio (with anonymized examples)
- Performed testing on real demo applications
- Strengthened analysis, reporting, and testing methodology skills
- Mentorship from senior QA engineers throughout the course
