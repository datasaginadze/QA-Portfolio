# Manual Software Testing

This folder contains QA-related certificates and achievements demonstrating completed training programs and recognised skills in Manual QA practices.  
All materials are shared in a generic or anonymised format to respect NDA and confidentiality agreements.

## Sample Certificate: Manual Software Testing Course
![Manual QA Certificate](../Assets/1course.jpg)

**Course Overview:**
- Instructor: Lela Gabelaia
- Sessions: Twice a week with additional special sessions
- Assessment: Control tests, group final project, individual oral assessment

**Topics Covered:**
- Software Testing fundamentals (SDLC, STLC, QA vs QC)
- Test case creation, execution, and test suites
- Bug lifecycle, reporting, and management (Jira, Trello)
- Checklists, traceability matrix, and test summary reports
- Website & mobile testing (cross-browser, responsive, I SLICED UP FUN, Play Store / App Store guidelines)
- API testing (Postman, Swagger)
- SQL basics for data validation
- UI/UX review and accessibility checks
- Introduction to automated testing (Selenium, Selenide, JUnit)
- Performance testing (JMeter basics, test execution)
